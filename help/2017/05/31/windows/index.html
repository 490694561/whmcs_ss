    一、下载Windows下的影梭软件

    最新版下载地址

    Win8系统下载地址

    Win7、XP、Vista系统下载地址

    如遇到以上软件都无法使用时可以在这里下载最新版影梭软件

    二、下载解压后运行影梭

    注意：如果XP下载后双击打不开，提示： 影梭 应用程序正常初始化失败。

    说明系统缺少.net framework。

    .NET Framework 2.0 版本，适合 Windows XP 用户

    .NET Framework 3.5 版本，适合 Windows 7 用户

    .NET Framework 4.0 版本，适合 Windows 8.1 用户

    请自行通过搜索引擎搜索下载。

    Windows8.1版本兼容微软最新发布的Windows10系统！

    三、首次运行，弹出编辑服务器窗口，请输入您在 影梭官网 购买的服务，正确填写服务器地址、端口、密码和加密方式，或者直接扫描二维码，然后点确定。

    点确定后，右下角弹出提示

    四、右键程序图标，弹出主菜单，勾选“启用系统代理”

    五、打开任意浏览器（IE/Chrome/Firefox等），即可开始科学上网
